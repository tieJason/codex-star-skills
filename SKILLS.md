# Skill 索引

## 工程

- `repo-cartographer`：仓库上手、架构地图、入口定位、实现路径规划。
- `bug-reproducer`：复现步骤、失败测试、回归覆盖、修复验证。
- `pr-review-sherpa`：聚焦真实 bug、回归风险和缺失测试的 PR 审查。
- `test-suite-hardener`：有意义覆盖、fixture 改进、flaky 测试治理。
- `api-integration-builder`：API client、webhook、重试、rate limit 和 typed error。
- `security-light-review`：鉴权、注入、secret、SSRF、XSS 和隐私风险轻审。
- `docs-from-code`：基于代码生成 README、runbook、API 文档和架构说明。

## 前端

- `frontend-polish-pass`：响应式布局、状态补全、视觉 QA 和可访问性基础检查。
