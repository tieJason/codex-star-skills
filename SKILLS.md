# Skill Index

## Engineering

- `repo-cartographer`: repository orientation, architecture mapping, implementation planning.
- `bug-reproducer`: reproduction steps, failing tests, regression coverage.
- `pr-review-sherpa`: senior-style PR review focused on bugs and missing tests.
- `test-suite-hardener`: meaningful coverage, fixture cleanup, flake reduction.
- `api-integration-builder`: external API clients, webhooks, retries, rate limits.
- `security-light-review`: practical security review for auth, injection, secrets, and data exposure.
- `docs-from-code`: READMEs, runbooks, API docs, architecture notes grounded in code.

## Frontend

- `frontend-polish-pass`: responsive UI refinement, states, visual QA, accessibility basics.
